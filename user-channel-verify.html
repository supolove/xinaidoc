角色登录校验文档

1. 请求地址

http://api.xhungame.com/api/user/check

请求方式 -POST

2. 请求参数

字段 类型 描述
uid Long 账号唯一标识
token String 用户登录返回的secretKey

3. 描述:

校验是否为该账号登录,防止串号

4. 注意:

账号校验登录有效时长为5分钟(5分钟之内没有校验,需要重新登录)

5. 返回结果:

code为0时, 则为在平台登录成功.其余code码返回错误信息

6.返回结果:

code为0返回结果: code为0时, 则为在平台登录成功.其余code码返回错误信息时, 则为在平台登录成功.其余code码返回错误信息
code message
0 账号认证成功
1100 账号未登录
1101 账号认证失败

请求成功:

{
    "code": 0,
    "content": "",
    "message": "账号认证成功",
    "success": true
 }

请求错误:

{
    "code":1101,
    "content":"null",
    "message":"账号未登录, ",
    "success": false"
}
