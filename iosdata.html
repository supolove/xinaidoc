iOS统计SDK接入文档

1、设置other linker flag

- 给Target中的other linker flag 添加 `-ObjC`

2、SDK功能接入

2.1 开启crash防御机制，自动收集上报crash信息

+ [XHCrashHandler share]

建议在 application:didFinishLaunching 中开启crash防御机制
例：

- (BOOL)application:(UIApplication *)application didFinishLaunchingWithOptions:(NSDictionary *)launchOptions {

    [XHCrashHandler share];
    return YES;
}

2.2 是否开启日志打印

+ (void)trackingPrint:(BOOL)isPrint;

2.3 统计功能初始化

+ (void)trackingInit:(XHDataSDKInitModel *)model;
参数 说明
model 统计初始化数据模型，模型中的每条属性都要赋值，没有值的需要赋值 XHParamUNKNOWN

参数说明

2.4 设备启动统计

+ (void)trackingStartup:(XHUserModel *)model;

2.5 注册统计

+ (void)trackingRegister:(XHUserModel *)model;

2.6 登录统计

+ (void)trackingLogin:(XHUserModel *)model;

2.7 登出统计

+ (void)trackingLogout:(XHUserModel *)model;

2.8 画像变更统计

+ (void)trackingProfile:(XHUserModel *)model;

2.9 角色升级统计

+ (void)trackingLevelup:(XHUserModel *)model;

参数说明

XHDataSDKInitModel

备注：该参数下的全部属性都是NSString类型

属性 说明
XH_appID App在游戏平台的AppID
XH_dataAppID App在统计平台的AppID
XH_versionPackage 包版本号
XH_channelID 渠道ID
XH_channelName 渠道名称

XHUserModel

备注：该参数下的全部属性都是NSString类型

属性 说明
XH_userID 账号标识(平台账号标识)
XH_userName 账号名称(平台账号名称)
XH_serverID 服务器ID
XH_serverName 玩家所在服务器名称
XH_vip Vip等级
XH_level 玩家等级
XH_roleID 角色标识（角色唯一标识)
XH_roleName 角色名称
XH_gender 角色所属性别(男，女)
