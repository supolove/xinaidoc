9、SDK功能接入

9.1 SDK初始化

- (void)SDKInitWithAppID:(NSString *)appID
                  appKey:(NSString *)appKey
               channelID:(NSString *)channelID
             channelName:(NSString *)channelName
          versionPackage:(NSString *)versionPg
              completion:(void (^)(BOOL result))handler;

参数说明:

参数 类型 说明
appID NSString 应用id/游戏id
appKey NSString 应用的Key
channelID NSString 渠道id
channelName NSString 渠道名称
versionPg NSString 包版本号
handler Block 初始化完成回调

回调说明:
result为YES则表示初始化完成；为NO则表示初始化失败，在初始失败的情况下请不要调用-showLoginWindow方法

9.2 SDK登录窗口呼出

- (void)showLoginWindow;

注意：初始化完成后再调用，如下：

XHSDK *sdk = [XHSDK shareSDK];
__weak typeof(sdk) weakSDK = sdk;
[sdk SDKInitWithAppID:@"AppID" channelID:@"渠道ID" channelName:@"渠道名称" completion:^(BOOL result) {

    if (result) {
        NSlog(@“初始化成功”);
        [weakSDK showLoginWindow];
    } else {
        NSlog(@“初始化失败”);
    }
}];

备注： 为防止循环引用使用weakSDK 调用 showLoginWindow

9.3 登录结果回调

@property (nonatomic, copy) void (^loginBlock)(BOOL result, NSDictionary *userInfo);

例：

sdk.loginBlock = ^(BOOL result, NSDictionary *userInfo) {
    if (result) {
        NSlog(@“登录成功”);
        NSString *secretKeyValue = userInfo[@"secretKey"];
        NSString *uidValue = userInfo[@"uid"];
    } else {
        NSlog(@“登录失败”);
    }
};

参数说明:

参数 说明
result 登入结果YES，表示登录成功；NO表示登录失败
userInfo 登录成功返回的用户信息，登录失败返回空字典

9.4 登出

- (void)logoutCompletion:(void (^)(BOOL result))handler;

例：

[sdk logoutCompletion:^(BOOL result) {
    NSString *logoutResult = result ? @"退出成功" : @"退出失败";
    NSLog(@"%@", logoutResult);
}];
参数 说明
handler 登出结果的回调，result为YES，表示登出成功；NO表示登出失败

9.5 切换账号的回调

@property (nonatomic, copy) void (^switchAccountBlock)(void);

例：

sdk.switchAccountBlock = ^{
    // Cp自定义切换账号后执行的内容
};

9.6 进入游戏服务器

- (void)loginGameServer:(NSString *)serverID
             serverName:(NSString *)serverName
               roleName:(NSString *)roleName
                 roleId:(NSString *)roleID
             completion:(void(^)(BOOL result))handler;

使用：登录成功后，即可调用该方法
例：

[sdk loginGameServer:@"gameServer" serverName:@"serverName" roleName:@"roleName" roleId:@"roleID" completion:^(BOOL result) {

    NSString *title = result ? @"角色登录成功" : @"角色登录失败";
    NSLog(@"%@", title);
}];
参数 说明
serverID 服务器ID
serverName 服务器名称
roleName 角色名称
roleID 角色ID
handler 进入游戏服务器的结果回调

9.7 商品购买

- (void)purchaseWithParams:(NSDictionary<PurchaseKey, NSString *> *)params
                completion:(void(^)(SKPaymentTransactionState state, NSError *error))handler;

注意： 参数params中包含9对键值对 例：

NSMutableDictionary *parameters = [NSMutableDictionary dictionary];
parameters[PurchaseKeyRoleID] = @"roleID";
parameters[PurchaseKeyRoleName] = @"name";
parameters[PurchaseKeyRoleLevel] = @"level";
parameters[PurchaseKeyServerID] = @"testID";
parameters[PurchaseKeyServerName] = @"testServer";
parameters[PurchaseKeyCpOrderID] = @"cpOderID";
parameters[PurchaseKeyCpProductID] = @"testCpProduct1";
parameters[PurchaseKeyCpExtension] = @"testCpExtension";
parameters[PurchaseKeyExtension] = @"extension";

[sdk purchaseWithParams:parameters completion:^(SKPaymentTransactionState state, NSError * _Nonnull error) {

    if (error) {
        NSLog(@"交易失败");
    } else {
        NSLog(@"交易成功");
    }
}];

(NSDictionary *)params 参数说明：

键 值
PurchaseKeyRoleID 角色ID
PurchaseKeyRoleName 角色名称
PurchaseKeyRoleLevel 角色等级
PurchaseKeyServerID 角色所在服务器ID
PurchaseKeyServerName 角色所在服务器名称
PurchaseKeyCpOrderID Cp订单ID
PurchaseKeyCpProductID Cp的商品ID
PurchaseKeyCpExtension Cp的扩展信息
PurchaseKeyExtension 预留的额外扩展信息
